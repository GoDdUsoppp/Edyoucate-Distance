# edYOUcate Website

## Overview

The edYOUcate website is a modern, responsive landing page developed for **edYOUcate**, an online and distance education consultancy. The website is designed to provide prospective students with information about online degree programs, showcase partner universities, and generate admission enquiries through an intuitive user experience.

---

## Project Purpose

This project serves as the official marketing website for edYOUcate and aims to:

- Present information about online and distance education
- Promote UGC-DEB approved universities
- Generate counselling and admission enquiries
- Provide students with information about available programs
- Improve online visibility through SEO optimization

---
